# 🌿 Paradise Nursery - e-plantShopping

## 📌 Project Overview

Paradise Nursery is an online plant shopping application where users can browse different categories of houseplants and add them to a shopping cart.

This project is built using **React + Redux Toolkit**.

---

## 🚀 Features

### 🌱 Product Listing Page

* Displays multiple categories of plants
* Each plant includes:

  * Image
  * Name
  * Price
* "Add to Cart" button:

  * Adds item to cart
  * Gets disabled after adding
* Navigation bar with:

  * Home
  * Plants
  * Cart
* Cart icon shows total items dynamically

---

### 🛒 Cart Page

* Displays all selected plants
* Shows:

  * Plant image
  * Name
  * Price
  * Quantity
* Features:

  * Increase / Decrease quantity
  * Remove item
  * Total cost calculation
* Buttons:

  * Continue Shopping
  * Checkout (Coming Soon)
